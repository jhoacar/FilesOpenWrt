INSTRUCCIONES DE USO

Preparacion USB

El dispositivo USB sera el encargado de distribuir la información, para ello es necesario contar con los archivos pertinentes para poder visualizarlos en pantalla, estos son:

  • 404.html

    Este es el archivo mas importante; en el contendra la redireccion al index.html

    Su contenido debe ser el siguiente: 

    
<head>
    <meta http-equiv="Refresh" content="0; URL=http://www.pagina.com">
<head>

    Es importante aclarar que este contenido debe estar en las primeras lineas del archivo, y que se puede cambiar la URL a cualquiera, pero que se incluya el protocolo http://, puede ser reemplazada, por ejemplo, por http://www.otrapagina.com.

  • index.html:

    Sera el encargado de mostrar la informacion que desea, por defecto viene acompañado con un script el cual se encarga de mostrar las imagenes encontradas en la carpeta pages y con el siguiente formato, modificable en el script.js:

    img+(numero de la imagen)+.jpg



    Contenido del script.js: 

                                    
const DIRECTORIO = "pages";
const NOMBRE = "img";
const EXTENSION = ".jpg";

const showImages = async function () {
    let numImage = 1;
    while (true) {
        const response = await fetch(DIRECTORIO + "/" + NOMBRE + numImage + EXTENSION);
        if (!response.ok) break;
        const dataImage = await response.blob();
        if (!dataImage) break;
        const objectURL = URL.createObjectURL(dataImage);
        const image = document.createElement("img");
        image.setAttribute("src",objectURL);
        document.getElementById("images").appendChild(image)
        numImage++;
    }
}

window.onload = showImages;


    Contenido del index.html: 

                                    
<html lang="es">
<head>
    <meta charset="utf-8" />
    <link rel="icon" href="/favicon.png">
    <meta name="viewport" content="width=device-width,initial-scale=1" />
    <meta name="theme-color" content="#000000" />
    <title>Menu</title>
    <style type="text/css">
        body{
            padding:0;
            margin: 0;
        }
        img {
            width: 100%;
            object-fit: contain;
        }
</head>  
<body>
    <div id="images"></div>
    <script type="text/javascript" src="script.js"></script>
<body>
